# Vicky's Restaurant & Lounge Website

A modern, elegant static website for Vicky's Restaurant & Lounge in Juba, South Sudan.

## Project Structure

This project is built using pure HTML, CSS, and JavaScript to ensure maximum compatibility with GitHub Pages and zero build-step requirements.

- `index.html`: Main entry point.
- `css/styles.css`: Custom styling and overrides.
- `js/main.js`: Core logic for menus, forms, and interactivity.
- `assets/`: Folder for images and icons (currently using Unsplash placeholders).

## Features

- **Responsive Design**: Fully responsive on Mobile, Tablet, and Desktop.
- **Dynamic Menu**: Filtering system for food and drink categories.
- **WhatsApp Reservation**: Direct integration with WhatsApp for table bookings.
- **Concierge Assistant**: A simulated AI assistant to recommend dishes.
- **Modern UI**: Styled with Tailwind CSS (CDN) and custom animations.

## Deployment Instructions (GitHub Pages)

1.  **Upload Code**: Push this entire repository to GitHub.
2.  **Settings**: Go to your repository **Settings** tab.
3.  **Pages**: Select **Pages** from the sidebar.
4.  **Source**: Under "Build and deployment", select **Deploy from a branch**.
5.  **Branch**: Select `main` (or `master`) and `/ (root)` folder.
6.  **Save**: Click Save.

Your site will be live in a few moments!
